# コミュニティーガイドライン

このガイドラインは、「MIMIさん全曲紹介」において、ユーザーのみなさんがコンテンツ（コメント、プロフィールなど）を共有・投稿する際のルールを定めたものです。
サイト上を居心地の良い場所に保つこと・他のユーザーが投稿されたコンテンツを見て不快な思いをしないことを目的としてルールを定めました。
このため、このことを意識すればルールに違反することはないはずです。逆に、このルールに記載されていない内容でも、目的を意識して良心に沿った行動を心がけてください。

## 宣伝について
ページの内容に関連したサイトのリンクは、自分のコンテンツか・収益化しているかなどにかかわらず、送信して問題ありません。
ただし、上記の「投稿してはいけない内容」を含むサイトのリンクは送信できません。

例：
- ✅「歌詞はここから見れます [https://w.atwiki.jp/hmiku/...](https://w.atwiki.jp/hmiku/pages/50621.html)」
- ✅「この曲の歌ってみたを投稿したので、良ければ見てみて [https://www.youtube.com/...](https://www.youtube.com/watch?v=xG6JKIp029Q)」
- ✖「アマギフ3万円分がもらえるサイト→ [https://...](https://www.youtube.com/watch?v=dQw4w9WgXcQ)」
- ✖「Tik tok風にアニメを作ってミマシタ!これはTik tokでも公開してるから!見てみて。（宣伝ダメでも送っているかも!ダメだったら教えてね☆）[https://...](https://www.youtube.com/watch?v=dQw4w9WgXcQ)」


## Botによるアクセス・投稿について
あらかじめ [お問い合わせ](/contact) から申請をお願いします。利用目的などを確認したうえで、柔軟に対応します。

- タイトルや公開日時などの、YouTube Data APIから取得されたデータをクローリングすることは、[YouTube API サービス ポリシー](https://developers.google.com/youtube/terms/developer-policies?hl=ja) に違反するため控えてください。（サービス側がデータを再配布しているとみなされる恐れがあるため）
- サイト側で設けている制限を回避することはしないでください。
