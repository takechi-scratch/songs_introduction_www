# ユーザーデータポリシー

- [利用規約](/docs/terms/service/)
- [プライバシーポリシー](/docs/terms/privacy/)
- [ユーザーデータポリシー](/docs/terms/userdata/)


[takechi](https://x.com/takechi_scratch)（以下、「製作者」といいます。）は、「[MIMIさん全曲紹介](/)」（以下、「本サービス」といいます。）の提供のため、ユーザーについての一定の情報（以下、「ユーザーデータ」といいます。）を収集します。クッキーを利用してユーザーデータを取得したり、個人情報を含むユーザーデータを収集することもあります。個人情報の取り扱い方法については、[プライバシーポリシー](/docs/terms/privacy/)をご確認ください。


## ログイン情報について
本サービスでは、[Firebase Authentication](https://firebase.google.com/products/auth?hl=ja)を使用して「Googleでサインイン」機能を提供しています。

- Google LLC及びその関係会社を通じて、個人情報である「名前」「メールアドレス」を取得します。非公開のユーザーデータにアクセスすることはなく、本サービスがユーザーに代わってGoogleアカウントを操作することはありません。
- サインイン機能を提供するため、Cookieが使用されることがあります。
- [Googleのセキュリティ設定ページ](https://myaccount.google.com/connections?filters=3,4)で、Googleアカウントへのアクセス権を削除することができます。

## YouTube APIの利用について
- 本サービスでは YouTube API を使用した機能を提供しています。
そのため、本サービスを利用すると、[YouTubeの利用規約](https://www.youtube.com/t/terms)に従うことに同意したとみなされます。
- YouTubeの機能を経由して収集されたユーザーのデータは、 [Googleのプライバシーポリシー](https://www.google.com/policies/privacy)に従って取り扱われます。
- YouTube APIを使用する際、ユーザーの認証情報は使用していません。

YouTube API を使用して提供している機能は次のとおりです。

### 動画データの取得
タイトル、公開時刻、動画の長さ、サムネイルのURLをバックエンドで取得し、[曲一覧](/songs/)や詳細情報ページで使用しています。
- 分析データには、YouTube Data APIから取得したものと、製作者が独自に作成したものが含まれています。APIを使用して取得したデータには「YouTube Data APIより取得」と明記しています。

### 動画の埋め込み
埋め込みを使用して、詳細ページから曲を再生できます。再生時に、YouTubeが自動でユーザーデータを収集することがあります。

### 再生リストの自動作成
曲一覧の検索結果から、再生リストを自動で作成できます。

- [公式チャンネル](https://www.youtube.com/@songs-introduction)のアカウントを使って再生リストを作成するため、ユーザーの認証情報は使用していません。
ただし、再生リストはチャンネル内で公開されるため、選択した曲の内容は誰でも見ることができます。

- 再生リストや、関連するデータを削除したい場合は、[お問い合わせ](/contact/)よりお気軽にご連絡ください。


## Google アナリティクスの利用について
本サービスでは、利用データや履歴の分析のため、Googleのアクセス解析ツール「Google アナリティクス」を使用しています。

- Google アナリティクスによって、以下の情報が収集されることがあります。
    - インターネット通信に通常用いられるシステム、デバイス、ネットワーク、通信に関する情報
    - 大まかな位置情報
    - サイト・アプリ上での行動に関するデータ
    - 閲覧ページに関するデータ
    - ユーザー識別子（Cookie、端末識別子等）
- 収集されたデータは、Google LLC及びその関係会社に送信されます。
- データは匿名で収集されており、個人情報は含まれていません。

詳細は[Google アナリティクス利用規約](https://marketingplatform.google.com/about/analytics/terms/jp/)や[Googleポリシーと規約ページ](https://policies.google.com/technologies/ads?hl=ja)をご覧ください。


## Cookieの拒否方法
ブラウザの設定を変更することにより、Cookieを無効にすることが可能です。ただし、Cookieを無効にした場合は、一部のサービスが利用できなくなる場合があります。
Cookieの設定変更方法については、各ブラウザのサポートサイトをご確認ください。
