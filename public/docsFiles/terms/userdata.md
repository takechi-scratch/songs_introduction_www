# ユーザーデータポリシー

- [プライバシーポリシー](/docs/terms/privacy/)

[takechi](https://x.com/takechi_scratch)（以下、「製作者」といいます。）は、「[MIMIさん全曲紹介](/)」（以下、「本サービス」といいます。）の提供のため、ユーザーについての一定の情報（以下、「ユーザーデータ」といいます。）を収集します。Cookieを利用してユーザーデータを取得したり、個人情報を含むユーザーデータを収集することもあります。製作者が個人情報を取扱う方法については、[プライバシーポリシー](/docs/terms/privacy/)をご確認ください。


## ログイン情報について
| 項目 | 詳細 |
| - | - |
| サービス名 | [Firebase Authentication](https://firebase.google.com/products/auth?hl=ja) |
| 収集する情報 | **名前**・**メールアドレス** （個人情報を含みます） |
| 情報の送信先 | Google LLC及びその関係会社 |
| Cookieの利用 | なし（管理者専用機能では利用する場合あり） |
| 利用目的 | 「Googleでサインイン」を使用した[ログイン](/login/)機能の提供のため |

- 非公開のユーザーデータにアクセスすることはありません。つまり、本サービスがユーザーに代わってGoogleアカウントで操作をすることはありません。
- [Googleのセキュリティ設定ページ](https://myaccount.google.com/connections?filters=3,4)で、Googleアカウントへのアクセス権を削除することができます。

### 専用チャンネル用アカウントの管理（管理者専用機能）
- YouTubeのデータ操作が可能なスコープで認証する際、Cookieを使用してログイン情報を管理します。

## YouTube APIの利用について
| 項目 | 詳細 |
| - | - |
| サービス名 | [YouTube API サービス](https://developers.google.com/youtube?hl=ja) |
| 収集する情報 | 再生リストを作成する際の動画の一覧・基本的なユーザーデータ（埋め込み再生時） |
| 情報の送信先 | Google LLC及びその関係会社 |
| Cookieの利用 | あり |
| 利用目的 | YouTubeを使用した機能の提供のため（以下で詳しく説明しています） |

- 本サービスを利用すると、[YouTubeの利用規約](https://www.youtube.com/t/terms)に従うことに同意したとみなされます。
- YouTubeの機能を経由して収集されたユーザーのデータは、 [Googleのプライバシーポリシー](https://www.google.com/policies/privacy)に従って取り扱われます。
- YouTube API を使用する際、ユーザーの認証情報は使用していません。

YouTube API サービス を使用して提供している機能は次のとおりです。

### 動画データの取得
タイトル、公開時刻、動画の長さ、サムネイルのURLをバックエンドで取得し、[曲一覧](/songs/)や詳細情報ページで使用しています。
- 分析データには、YouTube Data APIから取得したものと、製作者が独自に作成したものが含まれています。APIを使用して取得したデータには「YouTube Data APIより取得」と明記しています。

### 動画の埋め込み
埋め込みを使用して、詳細ページから曲を再生できます。再生時に、YouTubeが自動でユーザーデータを収集することがあります。

### 再生リストの自動作成
曲一覧の検索結果から、再生リストを自動で作成できます。

- [公式チャンネル](https://www.youtube.com/@songs-introduction)のアカウント（認証情報）を使って再生リストを作成します。
ただし、再生リストはチャンネル内で公開されるため、選択した曲の内容は誰でも見ることができます。

- 再生リストや、関連するデータを削除したい場合は、[お問い合わせ](/contact/)よりお気軽にご連絡ください。


## Google アナリティクスの利用について
| 項目 | 詳細 |
| - | - |
| サービス名 | [Google アナリティクス](https://developers.google.com/analytics?hl=ja) |
| 収集する情報 | 以下に記載しています |
| 情報の送信先 | Google LLC及びその関係会社 |
| Cookieの利用 | あり |
| 利用目的 | 利用データや履歴の分析 |

- Google アナリティクスによって、以下の情報が収集されることがあります。
    - インターネット通信に通常用いられるシステム、デバイス、ネットワーク、通信に関する情報
    - 大まかな位置情報
    - サイト・アプリ上での行動に関するデータ
    - 閲覧ページに関するデータ
    - ユーザー識別子（Cookie、端末識別子等）
- 収集されたデータは、Google LLC及びその関係会社に送信されます。
- データは匿名で収集されており、個人情報は含まれていません。

詳細は[Google アナリティクス利用規約](https://marketingplatform.google.com/about/analytics/terms/jp/)や[Googleポリシーと規約ページ](https://policies.google.com/technologies/ads?hl=ja)をご覧ください。

## Cookieの拒否方法
ブラウザの設定を変更することにより、Cookieを無効にすることが可能です。ただし、Cookieを無効にした場合は、一部のサービスが利用できなくなる場合があります。
Cookieの設定変更方法については、各ブラウザのサポートサイトをご確認ください。
