# 利用規約・プライバシーポリシー

> この規約は現在暫定的なものです。将来的に変更される可能性があります。

## YouTube APIの利用について
- 本サービスでは YouTube API を使用した機能を提供しています。
そのため、本サービスを利用すると、[YouTube の利用規約](https://www.youtube.com/t/terms)に従うことに同意したとみなされます。

- YouTubeの機能を経由して収集されたユーザーのデータは、 [Googleのプライバシーポリシー](https://www.google.com/policies/privacy)に従って取り扱われます。

YouTube API を使用して提供している機能は次のとおりです。

### 動画データの取得
タイトル、公開時刻、動画の長さ、サムネイルのURLを取得し、[曲一覧](/songs/)や詳細情報ページの中で使用しています。
- 分析データには、YouTube Data APIから取得したものと、製作者が独自に作成したものが含まれていますが、
APIを使用して取得したデータには「YouTube Data APIより取得」と明記しています。

- データはバックエンドで自動で更新しており、取得するためにユーザーデータ（認証情報など）は使用していません。

### 動画の埋め込み
埋め込みを使用して、詳細ページから曲を再生できます。
- 動画の再生時に、YouTubeが自動でユーザーデータを収集することがあります。

### 再生リストの自動作成
曲一覧の検索結果から、再生リストを自動で作成できます。

- [公式チャンネル](https://www.youtube.com/@songs-introduction) のアカウントを使って再生リストを作成するため、ユーザーの認証情報は使用していません。
ただし、再生リストはチャンネル内で公開されるため、選択した曲の内容は誰でも見ることができます。

- 再生リストや、関連するデータを削除したい場合は、[お問い合わせ](/contact/)よりお気軽にご連絡ください。

#### 認証情報の管理について（技術的な内容）
- 公式アカウントの認証情報は、製作者が手動で発行し、環境変数として安全に保管しています。
- 公式アカウントは[Googleのセキュリティ設定ページ](https://myaccount.google.com/connections?filters=3,4)でアクセス権を削除することもできます。


## Googleアナリティクスの利用について
本サービスでは、Googleのアクセス解析ツール「Googleアナリティクス」を使用しており、データの収集のためにCookieを使用しています。

- データは匿名で収集されており、個人を特定するものではありません。
- Cookieを無効にすることで収集を拒否することができます。お使いのブラウザの設定をご確認ください。

この規約に関しての詳細は[Googleアナリティクス利用規約](https://marketingplatform.google.com/about/analytics/terms/jp/)や[Googleポリシーと規約ページ](https://policies.google.com/technologies/ads?hl=ja)をご覧ください。

## 準拠法・裁判管轄
- 本規約の解釈にあたっては、日本法を準拠法とします。
- 本サービスに関して紛争が生じた場合には、製作者の居住地を管轄する裁判所を専属的合意管轄とします。
